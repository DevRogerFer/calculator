## 📖 Sobre o Projeto

Este projeto é uma calculadora simples, direta e focada em boa experiência de uso.

Com estrutura enxuta e sem frameworks, ela demonstra um fluxo completo de lógica aritmética no front-end:

- Operações de adição, subtração, multiplicação e divisão
- Formatação de números com separador de milhar
- Controle de estado (AC e DEL)
- Layout adaptado para desktop e mobile

## ✨ Destaques do Projeto

Mesmo sendo um projeto pequeno, entrega funcionalidades práticas para uso real:

- Zero complexidade de framework
- Código leve e fácil de entender
- Formatação de números em tempo real
- Organização clara entre estrutura, estilo e lógica
- Responsividade com folha de estilo separada
- Hover restrito a dispositivos com ponteiro (sem travamento em touch screens)

## 🧠 Tecnologias Utilizadas

- **HTML5** — estrutura semântica dos botões e display
- **CSS3** — estilos, layout em grid e estados visuais
- **JavaScript (Vanilla)** — lógica da calculadora, formatação e controle de display
- **Google Fonts (Roboto)** — tipografia da interface
- **CSS Grid** — posicionamento responsivo dos botões
- **Media Queries** — ajustes de layout para telas menores

## ⚙️ Funcionalidades

- ✅ Inserir números e decimais
- ✅ Realizar operações: `+`, `-`, `*`, `÷`
- ✅ Encadear operações sequencialmente
- ✅ Apagar o último dígito (DEL)
- ✅ Limpar tudo (AC)
- ✅ Formatar números com separador de milhar no display
- ✅ Responsividade em dispositivos móveis
